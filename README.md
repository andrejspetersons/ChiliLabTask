Simple Product API fetching app </p>

## Installation
+ Install zip file
+ Unpack in fold
+ Open in VS Code terminal
+ Move to the ProductListFetch
```
cd ProductListFetch
npm install //install required dependencies
npm run dev
```
[Application runs on http://localhost:5173](http://localhost:5173)</p>
